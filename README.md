# aishort — Cursor skill: short-term quant + AI (research only)

Educational Cursor Agent Skill for **short-horizon** equity discussion using common **technical / quant-style** indicators plus **AI synthesis** of conflicting signals. **Not** investment advice; **no** trade execution.

## Install (Cursor)

**Personal (all projects):** copy the folder `stock-short-term-quant` into:

- Windows: `%USERPROFILE%\.cursor\skills\`
- macOS / Linux: `~/.cursor/skills/`

**Single project only:** copy into:

`<your-repo>/.cursor/skills/stock-short-term-quant/`

Restart Cursor or reload the window if skills do not appear.

## Contents

- `stock-short-term-quant/SKILL.md` — the skill

## Publish to GitHub

1. Create a new empty repository on GitHub.  
2. `cd` to this `aishort` folder and run `git init`, `git add .`, `git commit`, `git remote add origin …`, `git push -u origin main`.  
3. Optional: add topics such as `cursor`, `cursor-skills`, `quantitative-finance`, `technical-analysis`.
